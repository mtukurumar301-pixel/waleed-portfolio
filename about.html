The short version

I'm Waleed — new to web development and working through freeCodeCamp's curriculum one certification at a time. I like that code gives you an immediate, honest answer: it either renders right or it doesn't.

Why I'm learning to code

I wanted a skill where progress is visible and testable. Every lesson ends in something that either works in the browser or doesn't — that feedback loop is what keeps me going.

How I work

Small commits, one concept at a time. I'd rather finish a plain layout that actually works than half-build something ambitious.

Currently

  • Working through freeCodeCamp's Responsive Web Design certification
  • Practicing flexbox and grid by rebuilding layouts from scratch
  • Writing this portfolio as my own personal project, not a template

Off-screen

When I'm not staring at a browser dev tools panel, I'm usually reading, tinkering, or trying to explain what I just learned to someone else — it's the fastest way I've found to tell if I actually understood it.
